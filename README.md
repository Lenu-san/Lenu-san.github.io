Site statique (HTML, CSS, JavaScript), bilingue français / anglais, sans dépendance ni framework.

## Personnaliser le lien LinkedIn

Ouvrir `script.js` et coller l'URL entre les guillemets :

```js
var LINKEDIN_URL = "https://www.linkedin.com/in/...";
```

Les boutons LinkedIn apparaissent automatiquement dès que l'URL est renseignée.
